# 🥗 HealthiBite

**HealthiBite** is a web application that provides **personalized health, diet, and fitness recommendations** based on user inputs.  
The system analyzes lifestyle information and suggests tailored plans to improve diet, manage basic health concerns, and support overall wellness.

---

## 🚀 Live Demo
Try the application here:

🔗 https://healthibite.onrender.com

---

## ✨ Features

- User-friendly interface for entering health and lifestyle information  
- Personalized recommendations for **diet, exercise, and sleep**  
- Suggestions tailored to **basic health conditions and wellness goals**  
- Responsive design compatible with **desktop and mobile devices**

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/healthibite.git
cd healthibite
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
flask run
```

Open your browser and go to:

```
http://127.0.0.1:5000
```
